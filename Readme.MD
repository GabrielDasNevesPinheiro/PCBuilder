## Classes and Objects

### Certificate
* `PLATINUM`
* `GOLD`
* `SILVER`
* `BRONZE`
* `BOMB`

### HardwareStatus
* `OFF`
* `WORKING`
* `BURNING`
* `MELTED`
* `NO_VIDEO`

### Component
* `status` -> `HardwareStatus`
* `name` -> `str`
* `brand` -> `str`

### Font < Component
* `power` -> `int`
* `certificate` -> `Certificate`

### Processor < Component
* `socket` -> `str`
* `clock` -> `int`
* `maxMemoryClock` -> `int`
* `cores` -> `int`
* `logicalCores` -> `int`
* `getTDP()` -> `int`

### RAM < Component
* `size` -> `int` < MB size, like 8192
* `clock` -> `int`
* `maxClock` -> `int`
* `getTDP()` -> `int`

### GPU < Component
* `coreClock` -> `int`
* `memoryClock` -> `int`
* `vram` -> `int` < MB size, like 4096
* `getTDP()` -> `int`

### Motherboard < Component
* `socket` -> `str`
* `processor` -> `Processor | None`
* `ramSlots` -> `int`
* `ram` -> `list[RAM | None]` < ram max lengh = ramSlots
* `gpu` -> `GPU`
* `getTDP()` -> `int`
* `attach(component)` -> `bool` < Can have overload
* `turnOn()` -> `void` < check status property on components.
* `__str__` -> `str` < Overwrite
